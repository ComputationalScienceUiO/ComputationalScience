Admission Criteria: Mechanics

  1. The program requires 80 ECTS within the basic mathematics and programming courses, equivalent to the University of Oslo mathematics courses MAT1100, MAT1110, MAT1120, MEK1100, MEK2200, INF1000/INF1100, MAT-INF3360.
  2. In addition, the program requires one of the following courses INF3331, MAT-INF3100, MAT-INF3xxx (Numerical analysis, new code) and/or MAT-INF3yyy (Dynamical systems, new code).
  3. An average mark C (European grading scale) is required for these courses.
